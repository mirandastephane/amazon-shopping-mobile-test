## Prerequisite
 - Python 3.11.7
 - Node.js and npm
 - Android Studio
 - Uiautomator2 2.29.4
 - Appium 2.4.1

## How execute

Download the dependencies:

```shell
  pip install Appium-Python-Client
```
Run one test Class:
```shell
python -m unittest ShoppingCartTest.ShoppingCartTest
```
